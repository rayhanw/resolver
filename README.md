# Resolver app [[Website](https://resolver.site)]

## An app to find the solution to your Rails (and Ruby) errors/problems by simply searching for your error message

### Stack

- Rails 7.0.3
- Ruby 3.1.1
- Docker (Compose)
- PostgreSQL
- TailwindCSS
- Makefile

### Deployments/Production

- Digital Ocean
- Dokku
- Docker
